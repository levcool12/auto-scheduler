# auto-scheduler
## Описание проекта

Auto Scheduler — это программа на Python для автоматической генерации учебного расписания для нескольких групп студентов.
Программа формирует расписание на основе входных данных, заданных в JSON-файлах.  

Входные данные включают:
- список учебных групп
- список преподавателей и их предметов
- список аудиторий
- список занятий с количеством часов

Алгоритм распределяет занятия по временным слотам, учитывая ограничения:
- преподаватель не может вести два занятия одновременно
- группа не может иметь два занятия одновременно
- аудитория не может быть занята двумя занятиями одновременно

Результатом работы программы является готовое расписание, которое представленно в виде удобной excel таблицы. 

## Структура проекта

- `data/` — папка с входными JSON файлами
- `auto_scheduler.py` — основной алгоритм генерации расписания
- `validator.py` — проверка корректности расписания
- `val.csv` — результаты проверки
- `24940-44schedule.xlsx` — пример готового расписания
- `requirements.txt` — список необходимых библиотек
- `README.md` — описание проекта

# Установка и запуск

## 1. Установить Python

Необходимо установить Python версии **3.9 или выше**.

Проверить установку можно командой:
```
python --version
```

## 2. Клонировать проект
```
git clone https://github.com/USERNAME/auto_scheduler.git
cd auto_scheduler
```

## 3. Установить необходимые библиотеки
```
pip install -r requirements.txt
```

## 4. запустить код
```
python auto_scheduler.py
```

## 5. открыть веб интерфейс
```
http://127.0.0.1:5000
```
## 6. загрузить .json файл и сгенерировать расписание
на ваш компьютер сохранится shcedule.xlsx с готовым расписанием
